## About The Project

**test4temp** is an experimental personality exploration platform that maps the classic "4 Temperaments" model onto intuitive weather metaphors. Originally developed as a university project and inspired by real-world assessment experiences, it serves as a technical demonstration of integrating AI engines into a modern full-stack web environment.

Rather than providing a clinical diagnosis, the system offers a metaphorical perspective on behavioral traits by categorizing them into: **Sunny** (Sanguine), **Stormy** (Choleric), **Rainy** (Melancholic), and **Cloudy** (Phlegmatic). It is designed as an engaging tool for self-reflection, powered by an AI engine that provides dynamic analysis based on qualitative user responses.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* **Software**:
  * [Node.js](https://nodejs.org/) (v18.x or higher)
  * [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
* **API Key**:
  * A valid **Groq API Key** for the AI Engine (get it at [Groq Cloud](https://console.groq.com/))

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Frynnn-69/test4temp.git
   cd test4temp
2. **Install dependencies:**
   ```bash
   npm install
3. **Setup environment variables: Create a .env.local file in the root directory and add your Groq API Key:**
   ```bash
   GROQ_API_KEY=your_api_key_here
4. **Run the development server:**
   ```bash
   npm run dev
**Open http://localhost:3000 with your browser to see the result.**

## Usage

Start your journey by simply entering your nickname and answering 15 interactive questions designed to reveal your behavioral patterns. Once completed, our AI engine processes your responses to build a comprehensive personality profile.

The key functionalities are as follows:
* Registration-Free Experience: Jump straight into the assessment without the friction of account creation.
* Privacy-First Data Persistence: Your test results are saved securely and privately in your browser's LocalStorage.
* Historical Tracking: Access and compare your last 3 assessment results instantly via the History tab.
* Dynamic Metaphorical Mapping: Receive a unique analysis that translates psychological data into evocative weather metaphors for easier self-reflection.
